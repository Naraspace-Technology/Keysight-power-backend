This project is Python FastAPI ASGI backend server for Keysight Power Supply (E36313A)

# Cloning this repository
```
git clone https://{your-token}@github.com/Naraspace-Technology/Keysight-power-backend
cd Keysight-power-backend
```

# Docker UP
```
docker-compose up -d --build
```

# Docker DOWN
```
docker-compose down
```

# Swagger(OPENAPI Document)
```
http://{your-host-ip}:8000/docs
```
